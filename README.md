# TicTacToe Squared
Tic Tac Toe, but bigger

## Requirement
- Docker (to build it)
- NodeJS (to develop it on yourself)

## How to run
1. Run `make build-all` to run the docker command
2. After successful built, run `make start` to start traefik, frontend and backend with the [compose.yaml](./compose.yaml)
3. Open the browser with [http://localhost:1337](http://localhost:1337)
4. Have fun with it

## TechStack

### Frontend / UI
- React
- React-Router
- React-i18next
- TailwindCSS
- MaterialUI
- MaterualUI-Icons
- socket.io-client

### Backend / Serverside
- Express (not really used)
- socket.io
